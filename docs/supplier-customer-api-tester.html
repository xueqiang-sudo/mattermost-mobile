供应商、客户管理 API 测试工具

供应商 / 客户列表 API

供应商和客户本质上是「公司」实体的两种类型，通过 type 字段区分。

type：supplier（供应商）| customer（客户）

以上默认 ID 会保存到本地，刷新页面后自动填充到各接口的对应输入框

公共端点
GET /health

健康检查

GET /

API 基本信息（name、version）

GET /api/v1/docs

API 端点列表（无需认证）

供应商管理
POST /api/v1/companies supplier

创建供应商（id 通常由客户端生成 UUID；需 owner_id）

GET /api/v1/companies supplier

获取所有公司（客户端过滤 type=supplier）

GET /api/v1/companies/:id

获取单个供应商

PUT /api/v1/companies/:id

更新供应商

DELETE /api/v1/companies/:id

删除供应商

DELETE /api/v1/companies/:id/force

强制删除供应商（含级联）

客户管理
POST /api/v1/companies customer

创建客户（id 通常由客户端生成 UUID；需 owner_id）

GET /api/v1/companies customer

获取所有公司（客户端过滤 type=customer）

GET /api/v1/companies/:id

获取单个客户

PUT /api/v1/companies/:id

更新客户

DELETE /api/v1/companies/:id

删除客户

DELETE /api/v1/companies/:id/force

强制删除客户（含级联）

响应 
点击上方按钮调用接口，响应将显示在此处
