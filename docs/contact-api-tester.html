GET /api/v1/companies/:id/departments

更新员工

删除员工

GET /api/v1/employees/:id/details

获取员工详细信息

员工-公司关联
POST /api/v1/employees/:id/companies

添加员工到公司

DELETE /api/v1/employees/:id/companies

从公司移除员工

GET /api/v1/employees/:id/companies

获取员工所属公司

GET /api/v1/employees/:id/companies/details

获取员工所属公司详情

GET /api/v1/company-employees/:companyId/employees

获取公司下所有员工

员工-部门关联
POST /api/v1/employees/:id/departments

添加员工到部门

DELETE /api/v1/employees/:id/departments

从部门移除员工

GET /api/v1/department-employees/:departmentId/employees

获取部门下所有员工

扩展接口（统计 / 部门树 / 员工 / 搜索 / 版本）

对应文档「新增功能」：总人数、移动部门、按父部门过滤部门列表、员工级联部门、祖先链、搜索、通讯录版本。

GET /api/v1/company-employees/:companyId/total-employees

公司总人数（去重）

GET /api/v1/department-employees/:departmentId/total-employees

部门总人数（含子部门，去重）

PUT /api/v1/employees/:id/move-department

移动员工部门：from_department_id、to_department_id、company_id

GET /api/v1/by_companies/:companyId/departments

公司部门列表；可选查询 parent_department_id：不传=全部，-1=仅顶级，≥0=某父部门子部门

GET /api/v1/employees/:id/cascade-departments

员工在指定公司的级联部门；必填查询参数 company_id

GET /api/v1/departments/:id/ancestors

部门祖先链（含自身，自上而下）

GET /api/v1/company-employees/:companyId/search

公司内搜索：keyword 必填；可选 department_id 限定部门及子部门

GET /api/v1/department-employees/:departmentId/search

部门范围内搜索：keyword、company_id 均为查询参数

GET /api/v1/versions/companies/:companyId/contacts

获取通讯录版本（客户端缓存）

PUT /api/v1/versions/companies/:companyId/contacts

手动刷新通讯录版本
