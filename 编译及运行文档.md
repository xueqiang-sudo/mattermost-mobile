# 编译及运行文档

## 环境
> 参考文档（Mattermost移动应用开发者设置）：https://developers.mattermost.com/contribute/more-info/mobile/developer-setup/
### 所需环境
* node 版本：22.11.0
* npm 版本：10.9.0
* ruby 版本: 3.2.0
* android studio 版本: 2025.2.3
* ios xcode 版本: 26.4
* watchman 版本: 2026.03.30.00
* android ndk 版本：27.1.12297006
* android sdk: 35.0.0


### 运行命令：
* 安装依赖：npm run clean-all && npm run install-pkg && npm run postinstall
* 运行命令本地运行:
  * android 运行: npm run android
  * ios 运行: npm run ios【ios 需要本地运行，需要在 https://developer.apple.com/account/resources/devices/list 上添加设备】
* 运行编译命令打包：
  * 编译 android: npm run rn-bundle:android && npm run build:android
  * 编译 ios: npm run rn-bundle:ios && npm run build:ios


### 上传到 TestFlight
> ios 编译及上传到 TestFlight 参考文件《ios 生产包编译及上传到 TestFlight 平台.docx》
* 按照上面的编译命令打包 ios 项目，得到 ipa 文件
* 在项目根目录执行上传 TestFlight 命令：bash scripts/upload-testflight.sh Dedalix_<版本号>.ipa
* 上传完成后，可以在 TestFlight 平台查看上传的版本信息：https://appstoreconnect.apple.com/apps/6779247503/testflight/ios
