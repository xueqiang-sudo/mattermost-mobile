# 编译及运行文档

## 环境
> 参考文档（Mattermost移动应用开发者设置）：https://developers.mattermost.com/contribute/more-info/mobile/developer-setup/
### 所需环境
* node 版本：22.11.0
* npm 版本：10.9.0
* ruby 版本: 3.2.0
* android studio 版本: 2025.2.3
* ios xcode 版本: 26.4
* watchman 版本: 2026.03.30.00
* android ndk 版本：27.1.12297006
* android sdk: 35.0.0


### 运行命令：
* 安装依赖：npm run clean-all && npm run install-pkg && npm run postinstall
* 运行命令本地运行:
  * npm run start
  * android 运行: npm run android
  * ios 运行: npm run ios
* 运行编译命令打包：
  * 编译 android: npm run rn-bundle:android && npm run build:android
    * 带签名：npm run rn-bundle:android && npm run build:android
    * 无签名：npm run rn-bundle:android && npm run build:android-unsigned
  * 编译 ios:
    * 带签名：npm run rn-bundle:ios && npm run build:ios【不能编译成功，需要等 ios 申请下来后再调试】
    * 无签名：npm run rn-bundle:ios && npm run build:ios-unsigned【能编译成功，但是安装不了】
