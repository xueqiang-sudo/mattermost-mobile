# 修改 select_team 页面实现企业选择功能

## 需求分析

需要修改 select_team 页面，使其显示类似企业微信的 UI，包含两个选项：
1. 加入企业 - 生成二维码
2. 创建企业 - 进入创建企业流程

同时需要移除现有的团队列表显示逻辑，因为加入企业必须通过二维码方式。

## 实现计划

### 1. 修改 select_team.tsx

- 移除团队列表相关的状态和逻辑（otherTeams, loading, joining 等）
- 简化组件结构，直接显示新的企业选择 UI
- 移除对 TeamList 组件的引用

### 2. 修改 no_teams.tsx

- 重命名为 enterprise_selection.tsx 或类似名称
- 实现新的 UI 布局，包含两个卡片式按钮
- 每个按钮包含图标、标题和操作文本
- 使用项目现有的样式系统和组件

### 3. 创建新组件

#### 3.1 create_team.tsx

- 创建企业的新组件
- 包含企业名称输入、创建按钮等
- 实现创建企业的逻辑

#### 3.2 join_team_qr.tsx

- 生成加入企业二维码的组件
- 包含二维码显示、用户信息等
- 实现二维码生成逻辑

### 4. 更新导航和路由

- 确保从企业选择页面可以正确导航到创建企业和加入企业二维码页面
- 处理返回逻辑

### 5. 样式和用户体验

- 确保所有组件符合项目的设计风格
- 使用现有的 Button 组件和样式系统
- 添加适当的动画和过渡效果
- 确保在不同设备上的响应式显示

## 技术要点

- 使用 React Native 的基本组件（View, Text, TouchableOpacity 等）
- 利用项目现有的主题系统和样式工具
- 实现简单的导航逻辑
- 确保代码符合项目的代码风格和规范
