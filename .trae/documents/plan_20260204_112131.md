# 重新设计加入企业的显示组件

## 需求分析

用户要求重新设计加入企业的显示组件，严格根据提供的图片进行设计，结合当前项目的配色风格进行微调。同时需要将生成二维码封装成一个公共组件，方便任意组件使用。

## 实现计划

### 1. 创建公共二维码生成组件

- 创建 `app/components/qr_code_generator/index.tsx` 文件
- 实现一个可复用的二维码生成组件
- 组件接受 `data` 属性，根据传入的信息生成对应二维码
- 组件接受 `size` 属性，可自定义二维码大小
- 组件接受 `logo` 属性，可选择是否显示 logo

### 2. 重新设计加入企业的显示组件

- 修改 `app/screens/select_team/join_team_qr.tsx` 文件
- 根据提供的图片进行设计，包含以下元素：
  - 标题：加入其他企业
  - 副标题：通过同事扫码确认身份加入
  - 二维码显示区域
  - 提示文字：同事扫码确认后，你可加入企业
  - 操作按钮：下载二维码到微信
- 结合项目的配色风格进行微调
- 确保只有二维码这一种加入方式

### 3. 实现下载二维码功能

- 添加下载二维码到本地相册的功能
- 添加分享到微信的功能
- 确保功能符合项目的设计风格

### 4. 测试和优化

- 测试二维码生成功能
- 测试下载和分享功能
- 确保在不同主题下的显示效果
- 优化用户体验和界面设计

## 技术要点

- 使用 React Native 的基本组件（View, Text, TouchableOpacity 等）
- 利用项目现有的主题系统和样式工具
- 实现二维码生成逻辑
- 实现文件下载和分享功能
- 确保代码符合项目的代码风格和规范
