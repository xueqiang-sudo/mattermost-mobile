# 创建图标展示组件

## 功能需求
- 创建一个组件，能够展示所有可用的 compass 图标
- 支持滚动条，方便查看大量图标
- 以网格形式展示图标，每个图标下方显示其名称

## 实现计划

### 1. 创建图标展示组件文件
- 在 `app/components/compass_icon/` 目录下创建 `IconGallery.tsx` 文件
- 该文件将包含一个新的组件，用于展示所有图标

### 2. 实现组件核心功能
- 导入 `CompassIcon` 组件和 fontello 配置文件
- 从配置文件中提取所有图标名称
- 使用 `FlatList` 或 `ScrollView` 实现滚动功能
- 使用网格布局展示图标，每个图标项包含图标和名称

### 3. 组件样式设计
- 使用 `View` 和 `StyleSheet` 创建网格布局
- 为图标项设置合适的大小和间距
- 为图标名称设置合适的字体大小和样式

### 4. 导出和使用
- 在 `app/components/compass_icon/index.tsx` 中导出新创建的 `IconGallery` 组件
- 提供使用示例，说明如何在其他地方导入和使用该组件

## 技术要点
- 使用 React Native 的 `FlatList` 组件实现高效的滚动列表
- 使用 `numColumns` 属性实现网格布局
- 从 fontello 配置文件中动态获取图标列表，确保组件能够自动包含所有可用图标
- 为组件添加适当的样式，确保美观和易用性
