## 重新设计创建团队屏幕计划

### 1. UI 重新设计
- 重构布局，提升美观度、大气感和高级感
- 添加欢迎区域，显示当前用户信息（例如："Hi, [用户名], 让我们创建您的企业"）
- 优化输入字段样式（边框、阴影等）
- 增强按钮设计，使其更加突出
- 使用浅色背景配色方案，确保整体视觉一致性

### 2. 添加 URL 输入部分
- 显示固定的 URL 前缀（不可编辑）
- 添加 URL 后缀输入字段
- 实现 URL 后缀验证逻辑：
  - 必须以字母开头
  - 只能包含小写字母、数字和连字符
  - 不能以连字符结尾
  - 提示用户使用简短易记的名称
- 显示验证错误信息

### 3. 功能增强
- 更新 `handleCreateEnterprise` 函数，将 URL 包含在 API 调用中
- 添加适当的错误处理
- 改进加载状态 UI

### 4. 样式更新
- 更新样式表以适应新布局
- 确保响应式设计考虑
- 与 Mattermost 设计系统保持视觉一致性

### 5. 实现步骤
1. 分析当前代码结构和功能
2. 添加用户信息显示功能
3. 实现 URL 输入和验证逻辑
4. 重构 UI 布局和样式
5. 更新表单提交逻辑
6. 测试功能和设计是否符合要求
