# 重新设计企业选择页面样式

## 目标
- 实现毛玻璃卡片效果
- 重新设计标题和副标题样式
- 保持整体背景色不变
- 整体设计大气上档次

## 实现方案

### 1. 导入必要的库
- 导入 `expo-blur` 库中的 `BlurView` 组件

### 2. 重新设计样式
- **容器样式**：保持现有的背景色，调整内边距和布局
- **标题样式**：使用更大的字体、更粗的字重，添加适当的间距
- **副标题样式**：使用更优雅的字体大小和颜色
- **卡片样式**：
  - 使用 `BlurView` 实现毛玻璃效果
  - 添加阴影效果增强层次感
  - 调整圆角和内边距
  - 优化按钮点击效果
- **图标样式**：调整图标大小和颜色，使其与整体设计协调

### 3. 代码修改
- 修改 `enterprise_selection.tsx` 文件
- 使用 `BlurView` 替换现有的 `View` 作为卡片容器
- 更新样式定义，实现新的设计效果
- 保持现有的功能逻辑不变

### 4. 预期效果
- 毛玻璃卡片效果，增强视觉层次感
- 现代化、大气的标题和副标题设计
- 保持与应用整体风格的一致性
- 提升用户体验和页面美观度
